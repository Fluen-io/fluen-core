Elements

Class

TemplateManager public

Manage the Jinja2 environment and render templates for documentation generation.

This class sets up template loaders, registers custom filters, and provides methods to render templates and get default context for template rendering.

Method

__init__ private

Initialize the TemplateManager with custom or default template directory and prepare the Jinja environment.

The initializer sets up the template loaders, configures the environment with autoescaping and block trimming options, and registers custom filters for templates.
_register_filters private

Register custom Jinja2 filters for use in templates.

This method adds custom filters to the Jinja environment such as 'format_type', 'anchor_id', and 'code_language'.
_detect_language private

Determine the programming language based on the file extension for syntax highlighting.

This method uses a mapping of file extensions to language names to detect and return the appropriate language for syntax highlighting in templates.
render_template public

Render a Jinja2 template using the provided context.

This method retrieves a template by name, renders it with the provided context, and returns the rendered template as a string. It logs an error and raises an exception if rendering fails.
get_default_context public

Provide a default context for templates based on the ProjectManifest.

This method generates a default context dictionary using information from the ProjectManifest, including project details, generation time, and git commit.

Variable

logger private

Log system for the TemplateManager to record errors and important events.

A logging.Logger instance initialized with the module's name for tracking operations and errors within the TemplateManager.
env private

Jinja2 environment instance for managing templates.

This instance of jinja2.Environment is configured with custom loaders and filter registrations to manage and render templates.

Purpose

This script is a command-line interface (CLI) tool for generating and exporting code documentation using a language model approach. It provides commands to generate documentation for a codebase and export it in different formats.

Elements

Class

FluenContext global

Holds the context information required for the CLI commands, such as configuration and console output settings.

Initializes with optional FluenConfig, default configuration path, a Rich console instance, and scan options.

Variable

pass_fluen_context global

A Click decorator that ensures the FluenContext instance is passed to CLI command callbacks.

Uses click.make_pass_decorator on FluenContext with ensure=True.

Function

cli global

Main command group for the CLI tool.

Provides the entry point for the CLI, handling configuration file setup and directory initialization.
docs global

Command group for documentation-related actions.

Serves as a holder for commands related to documentation generation and export.
generate docs

Generates documentation for the specified codebase.

Handles initialization of scan options, execution of the documentation generation process through an orchestrator, and reporting progress.
run_generation generate

Async function to run the documentation generation process.

Uses Rich's Progress to show progress of the documentation generation, reporting success or failure.
export docs

Exports generated documentation to the specified format and location.

Validates if documentation manifest exists, initializes necessary components, and performs the export if possible.
run_export export

Async function to handle the exportation process of the documentation.

Coordinates the export process using documentation generator based on the selected output format and directory.
